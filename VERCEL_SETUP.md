# Desplegar LuloPy en Vercel gratis (plan Hobby)

> Vercel Hobby es gratis para **uso personal/no comercial**. Revisa sus políticas si planeas uso comercial.

## 1) Crear cuenta
1. Entra a https://vercel.com
2. Sign up con GitHub (recomendado).
3. Asegúrate de quedarte en el plan **Hobby**.

## 2) Importar el repo
1. Dashboard → **Add New...** → **Project**
2. Importa tu repositorio `lulopy`.
3. Configuración:
   - Framework Preset: **Other**
   - Build Command: vacío
   - Output Directory: vacío
4. Clic en **Deploy**.

## 3) Probar
- Abre la URL `*.vercel.app` que te da Vercel.
- Confirma que “Python” carga y puedes ejecutar un ejercicio de código.

## 4) Actualizar
- Cada vez que hagas commit/push a GitHub, Vercel redeploya automáticamente.
