# Cómo crear el repositorio en GitHub y subir LuloPy (sin costo, sin usar git)

## 1) Crear el repositorio
1. Entra a https://github.com e inicia sesión.
2. Arriba a la derecha: **+** → **New repository**.
3. Nombre sugerido: `lulopy`
4. Visibilidad: **Private** (recomendado) o Public.
5. NO marques “Add a README” (ya viene uno).
6. Clic en **Create repository**.

## 2) Subir el contenido (desde navegador)
> Importante: GitHub **no** “descomprime” zips automáticamente al subirlos.
> Debes **descomprimir** el zip en tu computadora y subir los archivos/carpetas.

1. Descarga el zip `LuloPy_GitHub_Upload.zip` y descomprímelo.
2. En la página del repo recién creado, clic en **Add file** → **Upload files**.
3. Arrastra **todos** los archivos y carpetas descomprimidas (incluyendo `index.html`, `app.js`, `practicas/`, etc.).
4. Espera a que termine el upload.
5. Abajo: “Commit changes” → clic en **Commit changes**.

✅ Listo: el repo ya contiene LuloPy.

### Tip rápido
Si tu navegador soporta “upload de carpetas”, arrastra la carpeta `practicas/` completa. Si no, arrástrala desde el explorador como carpeta (Chrome/Edge suele funcionar).

## Alternativa (recomendada si vas a actualizar seguido): GitHub Desktop
1. Instala GitHub Desktop (gratis).
2. File → Clone repository → elige tu repo.
3. Copia estos archivos dentro de la carpeta clonada.
4. Commit → Push.
